# 中间优化版本 V2 需求文档

## 版本定位

中间优化版本的目标是把初生版本从“核心可用”推进到“日常主力绘画工具”。

这个阶段不追求最终形态的所有高级能力，而是重点强化：

- 稳定器和笔刷表现。
- 图层和快照工作流。
- Time-lapse 绘画过程记录。
- 更聪明的辅助绘制。
- 更完整的文件交换能力。
- 更稳定的大画布体验。

## 核心目标

- 让用户可以长期使用软件完成正式作品。
- 让快照成为比普通历史更有价值的创作节点系统。
- 让 Time-lapse 成为自然、轻量、不影响绘画的分享功能。
- 让辅助功能按需出现，用完收起。

## 性能优化

### 第一优先设备

中间版本的体验优化仍以用户当前真实绘画环境作为第一验证环境：

- MacBook Air，Apple M5，10 核 CPU，24GB 内存，macOS 26.4.1。
- Wacom 路线优先。
- 当前临时测试板：Wacom 影拓 5 / Intuos5。
- 后续重点测试板：Wacom Intuos Pro M / PTH-660。

如果后续扩展到其他设备，不能牺牲这套第一优先设备上的绘画体感。所有稳定器、笔刷、水彩、涂抹、快照和 Time-lapse 优化都应先在这套设备上确认。

同时保留 macOS 10.12 作为旧系统兼容目标，但它不是第一开发基准。中间版本允许对旧系统降级：

- 旧系统优先保证基础绘画、图层、保存、导出和压感。
- Time-lapse、高级水彩、复杂缓存管理、PSD 兼容和 UI 布局功能可按系统能力减少。
- 不为了旧系统兼容牺牲第一优先设备和现代 macOS 上的笔刷延迟与画布流畅度。

### 大画布

继续巩固以下目标：

- `6000 x 6000` 以下：`150-300` 常规图层顺滑。
- `8000 x 8000` 以下：`50-100` 常规图层顺滑。

优化方向：

- tile 缓存淘汰策略。
- 图层合成缓存。
- 缩略图后台生成。
- 大笔刷局部更新。
- 空白 tile 跳过。
- 低内存状态提示。
- 磁盘缓存预算管理。

### 缓存管理

加入设置项：

- 历史和缓存最大占用：20GB / 50GB / 100GB / 自定义。
- 自动清理普通历史缓存。
- 手动快照默认不自动删除。
- 用户可查看当前文件历史占用。

## 笔刷增强

### 稳定器增强

优化初生版本的稳定器，形成更完整的 SAI-like 手感。

增强点：

- 自适应速度平滑。
- 曲率保护。
- 快速长线增强稳定。
- 慢速刻线减少干预。
- 压感变化平滑。
- 起笔和收笔自然过渡。

界面仍保持简单：

- 稳定器强度。
- 自然修线。
- 速度响应。
- 压感曲线。

### 笔刷库

增强默认笔刷：

- 线稿笔多种变体。
- 铅笔多种颗粒。
- 软喷枪和硬喷枪。
- 马克笔。
- 水彩笔。
- 混色笔。
- 涂抹笔。

### 水彩和混色

中间版本需要让水彩成为可用主力工具。

参数：

- 混色。
- 水量。
- 稀释。
- 残留。
- 纸纹强度。
- 边缘沉积轻量模拟。
- 压感控制水量/浓度。

不做昂贵真实流体模拟，仍以视觉模拟和响应速度为优先。

### 涂抹增强

增强基础涂抹：

- 采样所有可见图层。
- 只涂抹当前图层。
- 沿笔触方向拉色。
- 保护透明像素。
- 纹理化涂抹。

## 快照系统增强

### 快照管理

完善快照面板：

- 快照缩略图。
- 快照名称。
- 快照备注。
- 创建时间。
- 锁定快照。
- 删除快照。
- 复制快照。
- 快照排序。

### 快照回退

保持规则：

- 点击快照完整回退。
- 回退前自动生成“回退前快照”。
- 回退行为进入特殊历史记录，防止误操作。

### 从快照复制内容

中间版本建议正式加入。

功能：

- 浏览快照图层列表。
- 从快照复制单个图层。
- 从快照复制多个图层。
- 从快照复制可见合成结果。
- 从快照复制选区范围内容，后续可做。

用户场景：

- 找回旧线稿。
- 找回旧配色。
- 对比不同阶段。
- 从旧版本中抽出某个图层继续使用。

### 快照和自动保存关系

继续保持分离：

- 自动保存只负责崩溃恢复。
- 快照是用户主动管理的创作节点。
- 自动保存不出现在快照列表。

## Time-lapse 绘画过程

### 功能定位

实现类似 Procreate 的绘画过程回放，不做实时屏幕录制。

核心原则：

> 绘画时记录轻量事件，导出时再后台渲染视频。

### 记录内容

- 笔画事件。
- 图层新增/删除/移动。
- 图层透明度和混合模式变化。
- 快照节点。
- 画布尺寸和背景。
- 关键画布视图变化，可选。

### 导出

支持：

- 30 秒。
- 60 秒。
- 3 分钟。
- 自定义时长。
- 1080p。
- 2K。
- 4K。
- 原画布比例。
- 16:9。
- 9:16。
- 1:1。

导出设置：

- 是否显示画布旋转/缩放。
- 是否显示背景色。
- 是否显示透明背景为纯色。
- 是否显示快照节点停顿。

## 辅助绘制

### QuickShape 增强

保持无弹窗、停驻触发。

增强：

- 更稳定的形状识别。
- 直线角度吸附。
- 正多边形识别。
- 修正后更好保留笔刷质感。
- 可调停驻触发时间。

### 智能闭合增强

增强填色体验：

- 闭合容差。
- 多图层参考。
- 当前图层参考。
- 指定参考图层。
- 抗锯齿填充。
- 扩展/收缩填充边缘。

### 对称辅助

中间版本建议加入。

支持：

- 左右对称。
- 上下对称。
- 四向对称。
- 径向对称。
- 对称轴移动。
- 对称轴旋转。
- 每图层独立启用。
- 显示/隐藏辅助轴。

对称辅助作为 overlay 层，不开启时不参与渲染。

### 基础透视辅助

加入较轻量的透视网格。

支持：

- 一点透视。
- 两点透视。
- 三点透视。
- 消失点拖动。
- 地平线调整。
- 图层启用透视吸附。

高级透视球、鱼眼、移轴放到最终版本。

## 文件交换

### PSD 支持

中间版本可以开始加入 PSD 兼容。

优先级：

1. PSD 导入。
2. 保留基础图层。
3. 保留图层透明度。
4. 保留常见混合模式。
5. PSD 导出放在后期或最终版本。

不必第一阶段支持 Photoshop 的全部高级特性。

### 其他格式

- TIFF 导出，可选。
- WebP 导出，可选。
- 剪贴板复制透明 PNG。

## 色彩增强

在第一版基础上加入：

- 色彩配置提醒。
- 导出时转换到 sRGB。
- Display P3 到 sRGB 转换预览。
- CMYK 预览作为实验功能。

仍不建议做完整 CMYK 绘制。

## 界面优化

### 面板管理

- 图层、颜色、笔刷、快照可停靠和折叠。
- `Tab` 一键隐藏所有面板。
- 支持保存工作区布局。
- 支持极简模式。

### 快捷键

- 快捷键自定义。
- 数位板按键映射。
- 临时工具快捷键。

不加入快捷圆盘菜单，保持电脑绘图软件的干净感。

## 稳定性和恢复

增强：

- 崩溃恢复列表。
- 自动保存间隔设置。
- 文件损坏恢复。
- 快照索引校验。
- 磁盘空间不足提醒。

## 中间版本暂缓内容

- 完整水墨笔刷系统。
- Perspective Orb 透视球。
- 鱼眼/广角/移轴透视。
- 人体 Pose Guide。
- 手部 Pose Guide。
- 完整 PSD 导出。
- 动画。
- 复杂文本和排版。
- 素材商店。

## 验收标准

- 软件可以作为日常主力绘画工具使用。
- `6000 x 6000` 主力画布下绘制、图层操作、快照都稳定。
- Time-lapse 可以完整导出几分钟绘画过程视频。
- 快照可以复制图层到当前作品。
- 对称和基础透视辅助不影响笔刷流畅度。
- 水彩、涂抹和混色达到可用于正式作品的水平。
